1) using getElementByTagName
Here we fetch the elements using the tagname,In this for even a single element an array is always created.
document.getElementByTagName("")
2) using getElementById
Here we fetch the elements using the id ,In this no array is created.
document.getElementById("")
3) using getElementsByClassName
Here we fetch the elements using the classname,In this for even a single element an array is always created.
document.getElementsByClassName("")
4) getElementsByName
Here we fetch the elements using the name,In this for even a single element an array is always created.
document.getElementsByName("")
